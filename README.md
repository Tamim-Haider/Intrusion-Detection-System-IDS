# Intrusion-Detection-System-IDS-
This project implements a comprehensive Intrusion Detection System (IDS) using "Snort", enhanced with custom-developed detection rules to identify and analyze various network-based threats in real time. The system continuously monitors live network traffic and provides immediate visual and logged alerts for activities such as port scans, brute-force attempts, malware communications, denial-of-service attacks, and protocol anomalies. Through a dedicated dashboard or alert console, administrators can observe live threat detection, view packet-level details, and respond to incidents as they occur. By employing both signature-based and heuristic methods via tailored rule sets, the IDS not only detects known attack patterns but also adapts to emerging threats, ensuring proactive network defense. The project highlights the practical application of open-source security tools in active monitoring, real-time alerting, and dynamic threat response within modern network environments.
